Atom est un IDE crée par GitHub donc avec une grande communauté il est gratuit et livré avec un gestionnaire de packages.

Ultra extensible, rapide, agréable à l'oeil, voici ce que Atom est !
Tous ces raccourcis sont personnalisables que cela soit ceux de Atom ou ceux des extensions (possibilité de créer son propre fichier et l’importer).
Les Packages sont faciles à installer, ils ont tous leur propre settings, qu’ils peuvent activer ou non.
Les thèmes, il existe de nombreux thèmes pour encore plus personnaliser l’apparence d’Atom.

Les Plus :

- Télétype
- Trés Personalisable
- Free
- Grande Communauté
- Intégration poussé de Git
- Split infini

Les Moins :

- Limite d'ouverture
- Logiciel jeune (Bugs)
- Pas de live preview
- Plugins instable pour certain
- Poids du Logiciel - Recherche Lente
